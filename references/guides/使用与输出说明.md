# 使用与输出说明

## 快速调用

```bash
python scripts/entrypoints/run_research.py \
  --query "比较材料中的关键指标并标出冲突" \
  --sources "资料目录" \
  --output "输出目录"
```

Windows PowerShell 可将命令写在同一行。`--sources` 与 `--url` 均可重复传入。

## 输出字段

- `overall_score`：216 项参考指标按 Excel 影响系数加权后的工程质量分，不代表事实真伪。
- `confidence_score`：工程质量分与证据相关性分的组合值。
- `evidence`：按相关性和多样性排序的证据片段，包含来源、定位、指纹和特征分。
- `conflicts`：数值、日期或肯否冲突；每项均保留双方来源。
- `gaps`：影响结论强度的信息缺口。
- `metrics`：每个启用指标的原始信号、得分、影响系数与加权贡献。

## 修改判断标准

只修改 `references/metrics/参考指标配置.xlsx` 的“参考指标”工作表：

1. 第一列“参考指标影响系数”允许 0-10 的数字。
2. “是否启用”填写“是/否”；停用后该指标不参与分母与分子。
3. 不要修改指标 ID；不要删除表头。
4. 修改后先运行 `python scripts/entrypoints/validate_config.py`。

运行程序不会缓存影响系数；每一次调用都会重新打开并读取 Excel。

## 失败处理

- 某个来源读取失败：写入告警，继续处理其他来源。
- Excel 配置错误：立即停止并报告具体行号，避免静默使用错误标准。
- 没有有效来源：生成低置信结果并提示补充材料。
- PDF 缺少解析依赖：跳过 PDF，其他格式继续；如需 PDF，可安装 `pypdf`。
